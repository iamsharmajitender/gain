## Key Layers

<Details className="gain-accordion" summary="Ingress & Gateway">
  The entry point for all user and system interactions. Handles authentication, rate limiting, request routing, and protocol translation before requests reach the G.A.I.N core.
</Details>

<Details className="gain-accordion" summary="G.A.I.N Core">
  The intelligence layer where grounding, adaptation, reasoning, and native execution happen. This is where RAG pipelines, agent runtimes, and LLM gateways live.
</Details>

<Details className="gain-accordion" summary="Data & Context Layer">
  Vector databases, knowledge graphs, relational stores, and file systems that provide the grounded context AI systems need. Data freshness and lineage are managed here.
</Details>

<Details className="gain-accordion" summary="Infrastructure Layer">
  Cloud compute, storage, networking, and platform services that host the entire stack. Designed for elastic scaling and multi-region resilience.
</Details>

<Details className="gain-accordion" summary="Observability & Governance">
  Spans all layers: capturing metrics, logs, traces, audit records, and policy enforcement. Not a layer you add later; it wraps the entire architecture.
</Details>
